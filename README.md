# Ticketmaster-Events
Utilizes Ticketmaster's and Google Maps API to show events in a local area. 

Check it out! https://adelgadillo314.github.io/Ticketmaster-Events/

Uses: 
- HTML
- CSS
- Javascript
- Bootstrap 
- TicketMaster API 
- Google Maps API 
- Ajax 
- Jquery
- form validation
- Google Firebase
